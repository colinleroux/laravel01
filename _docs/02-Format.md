# Formats

The Format lookup table.

Includes the format of the book/item.

## Table Design

| Field Name  | Type     | Size | Options                 | Default |
|-------------|----------|------|-------------------------|---------|
| id          | big int  | -    | unsigned, autoincrement |         |
| name        | string   | 64   |                         |         |
| description | string   | 255  | nullable                |         |
